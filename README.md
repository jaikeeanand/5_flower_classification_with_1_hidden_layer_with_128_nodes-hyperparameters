# 5 Flower Classification with 1 Hidden Layer and 128 Nodes - Hyperparameters

This repository demonstrates a flower classification model using a single hidden layer with 128 nodes. The implementation has been optimized for Google Colab and provides various hyperparameters to tune the model's performance.

## Getting Started

To get started, clone the repository and open it in Google Colab for execution.

### Prerequisites

- Python 3.x
- Required libraries:
    - TensorFlow
    - Keras

### Instructions

1. Open the notebook in Google Colab.
2. Download the dataset.
3. Follow the instructions outlined in the notebook to train and evaluate the model.
